# 功能

- 在 Obsidian 的 Markdown 文档中嵌入 Excel 文件，并显示表格内容。

# 用法

- 嵌入 Excel 文件：
  - ![[workbook.xlsx#worksheetname!a1:z55|1000*1000]]
  - ![[Excel 文件名.xlsx#工作表名!引用区域|宽*高]]
